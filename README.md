# Alma Tattoo - Landing Page

Projeto one-page de estúdio de tatuagem feito com **Vite + React + TypeScript + Tailwind CSS**.

## Rodando localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy no GitHub Pages

Este projeto já está configurado com `base: '/teste-vite-codex/'` no Vite e scripts de deploy.

```bash
npm run deploy
```

> Se o nome do repositório for diferente de `teste-vite-codex`, ajuste o `base` em `vite.config.ts`.
